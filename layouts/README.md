# 布局目录

**用于存放应用的布局组件**

### 相关注意事项

- 若无额外配置，该目录不能被重命名

- 可以通过编辑 layouts/error.vue 文件来定制错误页面

- 定制默认的 html 模板，只需要在应用根目录下创建一个 app.html 的文件
