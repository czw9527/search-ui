# 组件目录

**用于存放常规 vue 组件**

### 相关注意事项

- 一般情况下，所有常规 vue 组件都分类存放在本目录下

- Nuxt 不会扩展这些常规 vue 组件

- 即这些组件不会像页面组件那样拥有 asyncData 等特殊方法
