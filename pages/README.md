# 页面目录

**Nuxt 依据本目录结构自动生成 vue-router 模块的路由配置**

### 相关注意事项

- 若无额外配置，该目录不能被重命名

- 要在页面之间使用路由，我们建议使用 nuxt-link 标签

- 页面组件实际上是 vue 组件，只不过 nuxt 为这些组件添加了一些特殊的配置项

- 特殊配置项：asyncData、fetch、head、layout、loading、transition、scrollToTop、validate、middleware
