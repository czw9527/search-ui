# 静态文件目录

**目录中的文件不会被 webpack 进行处理**

### 相关注意事项

- 若无额外配置，该目录不能被重命名

- 该目录下的文件会映射至应用的根路径 / 下

- 像 robots.txt 或 sitemap.xml 这种类型的文件就很适合放到本目录中
