## 启动项目

``` bash
# 前置条件：安装 Node.js 及 Yarn 包管理工具

# 安装依赖
$ yarn install

# 启动热部署开发模式
$ yarn dev

# 编译代码及启动生产环境服务器
$ yarn build
$ yarn start

# 生成静态应用部署文件
$ yarn generate
```
