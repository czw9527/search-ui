# 状态树目录

**使用状态树来管理状态十分有必要**

### 相关注意事项

- 若无额外配置，该目录不能被重命名

- 目录下的每个 js 文件都会被转换成为状态树指定命名的子模块

- 注意为了避免返回引用类型并导致多个实例相互影响，state 的值应该始终是函数
